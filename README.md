# Image Slider Project

## Description
This project is a responsive and interactive image slider built using HTML, CSS, and JavaScript with the Swiper.js library.  
It includes smooth transitions, autoplay functionality, pagination indicators, navigation arrows, background music, and particle effects.

## Features
- Smooth animated image slider
- Automatic slideshow (every 3 seconds)
- Loop back to first image
- Navigation arrows and pagination indicators
- Responsive design for desktop and mobile
- Background music with play/pause control
- Particle background animation

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Swiper.js
- Particles.js

## How to Run
1. Download or extract the project ZIP file.
2. Open the `index.html` file in any modern web browser.
3. Use arrows or pagination dots to navigate slides.

## Usage
- Click arrows to move slides.
- Slides change automatically every 3 seconds.
- Click play button to start background music.
